# Daily Expense Tracker - Java Console App

## Project Description
A console app for students to manage daily expenses.  
- Set a daily budget and minimum keep.  
- Track expenses per day.  
- See daily and weekly spending summaries.  
- Prevent overspending below minimum balance.

## Features
- Add daily expenses (item name & price).  
- Auto-calculate Current Money, Daily Spent, Weekly Spent, Weekly Saved.  
- Simple console interface.

## Language
- Java (Scanner for input)

- `DailyExpense.java` → Main logic
